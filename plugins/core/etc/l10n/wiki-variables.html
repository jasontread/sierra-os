The following variables anywhere in your markup to their corresponding values. This substitution occurs every time that the markup is saved. The variable names are case-sensitive.
  • ~~~: the user's name (same as {$name})
  • ~~~~: the user's name and date (same as {$name} {$date})
  • {$name}: the user's name
  • {$userName}: the user's username
  • {$email}: the user's email
  • {$thumbnailUri}: the absolute uri to the user's thumbnail picture (or to the "No Image Available" image if the user does not have a picture). This will result in the Wiki image processing converting the uri to an img element (unless the variable name is enclosed in double brackets - i.e. [[{$thumbnailUri}]])
  • {$date}: the current date
  • {$datetime}: the current date and time
  • {$serverUri}: the uri to the server
